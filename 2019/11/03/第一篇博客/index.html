第一篇博客

发表于 2019-11-03   |   分类于 前端

写在前面

这篇博客将会详细的讲述如何快速利用hexo和hexo的next主题搭建一篇博客

第一部分

文件安装和配置部分:

  • 1. node安装：

    首先你需要进入到node.js的官网下载当前推荐的稳定版本的node。

    下载完成后就安装node

  • 2. github注册

    进入github网站注册自己的个人github账号。

  • 3. 创建特定的仓库

    点击github右上角新建一个仓库

    Repository必须按照 username.github.io形式进行创建

    之后点击Creat repository，仓库就创建成功了！

  • 4. 安装git环境

    windows ：Git for Windows
    mac : Git for Mac

  • 5. github SSH 配置

    1. 打开终端 输入命令 ssh-keygen -t rsa -C”github注册时的邮箱”
    2. 输入passphrase
    3. 前往终端显示的文件夹寻找.ssh 此文件夹由于以.开头所以此文件夹为隐藏文件夹。必须按下command+shift+.才能显示文件夹
    4. 打开id_rsa.pub文件复制文本
    5. 进入github个人用户点击Setting后再点击SSH and GPG keys 将密钥粘贴上去并点击 Add SHH key

  • 6.hexo 本地使用

    1. 新建一个文件夹专门存放博客

    2. 安装hexo
      在终端实现此命令

    cd 你的文件夹
    npm install hexo-cli -g

    1. 初始化博客
      终端实现命令
      hexo install 你的博客名字

    2. 安装文件
      终端实现命令
      npm install

    3. 本地访问博客
      终端实现命令
      hexo g (保存你对博客编辑)
      hexo s -p 5555 (这是个域名，应该可以随便写)

    然后你就可

  • 7. 安装sublime Text

    Sublime Text 是一个代码编辑器，可以用来编辑你的博客。进入官网进行下载安装就好了。

  • 8. hexo 发布至github

    1. 更改站点配置文件

    2. 终端实现命令
      hexo npm install –save hexo-deployer-git 安装依赖
      hexo g
      hexo d (上传博客到github)

    记得上传之前一定要先保存你的编辑，一般直接关闭Sublime Tex†，勾选save就可以保存编辑

  • 9. 安装next主题

    1. 下载安装next
      前往Next 发布页面 下载安装稳定版本
    2. 配置文件
      在hexo根配置文件 _config.yml 中设置你的主题
      theme： next
    3. 导入next
      将下载好的next放进你存放博客的文件夹的theme文件夹里

第二部分

hexo基本配置

  • 1. 更改next主题为中文

    进入主题配置文件(即next主题下的_config.ytml,你博客文件下的为站点配置文件)将language改为zh-Hans

  • 2. 添加标签页和分类页

    1. 进入主题配置文件,搜索menu。再将menu下的tags和categories打开。
    2. 进入终端输入命令
      hexo n pag tags
      hexo n pag categories
    3. 输入命令后会发现你的博客文件夹里的source下多出了两个tags和categories文件夹，此时分别进入两个文件夹里的index.md文件。将title改为分类和标签,加上type。

  • 3. 更改next主题样式

    进入主题配置文件搜索schemes可以看到有三张样式,选择一种你喜欢的样式。
    1. Muse样式
    2. Mist样式
    3. Pisces样式

  • 4. 更改头像

    1. 将头像图片放入next/source/images
    2. 进入主题配置文件搜索avatar,在avatar后加入/images/你图片的名字

  • 5. 头像改为圆形旋转

    效果图:

  • 6.新建一篇文章

    1. 终端输入命令
      hexo n 文章名字
      这时候就会创建一篇md格式的文章在/source/posts中
      进入这篇文章使用markdown语法就可以写文章啦!
    2. 为文章添加标签和分类
      进入你的文章,在开头进行添加分类和标签
